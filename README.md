# ✅MicraJava版で、プレイヤーがSpigotにログインした時に、青いベッドを1個を持っている事とベッドに入ろうとした時に持っているアイテムが最大64個所持している。






![スクリーンショット 2025-02-25 210427](https://github.com/user-attachments/assets/90817279-ce81-4eed-a2b6-958af9d1d635)












## 📌内容



### 1.プレイヤーがSpigotにログインした時に、青いベッドを1個を持っている



### 2.ベッドに入ろうとした時に持っているアイテムが最大64個所持している。(※マイクラJava版では、アイテム所持数が最大64個までしかできない。)



## 📝利用方法

### Windowsの場合


###  1. マイクラJava版にログインするとプレイヤーの空の持ち物アイテムに「青いベッド(🛏)」が一個持っている状態になる方法

#### マイクラJava版にログインしたら、最初にマルチプレイをクリックする。




![スクリーンショット 2025-03-09 060613](https://github.com/user-attachments/assets/6010bf6e-8608-4274-87cb-180af06ab16b)





#### 次に「Micraft_Server」をクリックする。






![スクリーンショット 2025-03-09 060630](https://github.com/user-attachments/assets/b088165c-ee72-468f-968a-e475654a74d9)






####  ログインすると空のホットバー(プレイヤーが持っているアイテムを入れる箱)に青いベッド(🛏)が一個所持されています。




![スクリーンショット 2025-03-09 060716](https://github.com/user-attachments/assets/6da71d61-0515-415a-9728-f7262b6b1bfe)




### 2.ベッドに入ろうとした時に持っているアイテムが最大64個所持している方法


#### ホットバー(プレイヤーが持っているアイテムを入れる箱)から青いベッド(🛏)を取り出して、右クリックで置いた後、青いベッドに対して、右クリックをもう一回押す事で、ホットバーのアイテム数が自動的に最大64個に増えるようになる。






![スクリーンショット 2025-03-09 060747](https://github.com/user-attachments/assets/fd902d4c-cbdf-4521-816e-c37187908343)
